== Description ==
= Introduction =

This easy to use Pal for Contact Form for WordPress site.

Each contact form can have its own PayPal settings. If a contact form with PayPal is enabled and the user sending the form, the email will be send as usual, then redirect to PayPal.

If you create problems, questions or comments about this PayPal plugins please a support request and we will quickly get back to you.

Configuring the following Contact=> PayPal Settings area to enter your PayPal settings details:

PayPal for Contact Form 7 interfaces with the Paypal sandbox to allow for testing. enable PayPal sandbox after You need to set up a developer test account for the sandbox at PayPal. Simply put you PayPal Sandbox Account in the Contact=> PayPal Settings admin panel.

= PayPal for Contact Form 7 Features =
*   Set item name, item price, item SKU.
*   PayPal currently supports 18 languages.
*   PayPal currently supports 25 currencies.
*   PayPal testing through SandBox.
*   PayPal return URL
*   PayPal Cancel URL

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "Pal for Contact Form" and click Search Plugins. Once you've found our plugin you can view details about it such as the the rating and description. Most importantly, of course, you can install it by simply clicking Install Now.

= Manual Installation =

1. Unzip the files and upload the folder into your plugins folder (/wp-content/plugins/) overwriting previous versions if they exist
2. Activate the plugin in your WordPress admin area.

= configuration =

Easy steps to install the plugin:

*	Upload "Pal for Contact Form" folder/directory to the /wp-content/plugins/ directory
*	Activate the plugin through the 'Plugins' menu in WordPress.
*	Go to Contact=> PayPal Settings.
